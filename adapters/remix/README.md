# Remix Adapter (Stub)

This is a placeholder for a future community adapter for [Remix](https://remix.run).

## Current Status

Not yet implemented. Contributions welcome!

## What it would need

- A `discover` implementation that walks Remix routes
- A thin runtime adapter for the chat endpoint
- Optional React wrapper for the widget
- Documentation + example app

See the [main Integration Guide](../../docs/integration.md) for the general shape of adapters.

## Interested?

Open an issue or PR! The core is deliberately designed to be easy to adapt.
